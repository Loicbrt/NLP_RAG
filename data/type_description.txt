De nombreux Pokémon possèdent deux types simultanément, ce qui modifie radicalement leurs faiblesses et résistances. Pour les Pokémon de type double, le calcul multiplie les coefficients de chacun de ses types. Cette mécanique peut créer des situations avantageuses ou catastrophiques selon les combinaisons.
Imaginons un combat contre un Pokémon Dragon/Vol comme Rayquaza. Les attaques de type Glace sont super efficaces contre le type Dragon ET contre le type Vol. Le multiplicateur atteint donc x4, infligeant des dégâts dévastateurs. Ce phénomène explique pourquoi certains Pokémon, malgré des statistiques impressionnantes, restent vulnérables face au type qui exploite leur double faiblesse.
À l’inverse, les deux types peuvent s’annuler mutuellement. Un Pokémon Eau/Dragon comme Hyporoi subit des attaques Plante avec un premier type sensible (Eau = x2) mais le second type résiste (Dragon = x0.5). Le résultat final donne un coefficient neutre de x1. Cette logique permet de construire des équipes équilibrées avec peu de faiblesses.
